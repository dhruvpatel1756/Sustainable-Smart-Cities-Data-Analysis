# 🌆 Smart City Citizen Activity – Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a Smart City Citizen Activity dataset to understand:

- Citizen lifestyle patterns
- Energy consumption behavior
- Transport usage impact
- Carbon footprint trends
- Health and sustainability indicators

The goal is to extract meaningful insights that can help in designing **sustainable urban policies**.

---

## 📊 Dataset Description

The dataset contains citizen-level activity and environmental data.

### 🔑 Features:

- `Citizen_ID`
- `Age`
- `Gender`
- `Mode_of_Transport`
- `Work_Hours`
- `Shopping_Hours`
- `Entertainment_Hours`
- `Home_Energy_Consumption_kWh`
- `Charging_Station_Usage`
- `Carbon_Footprint_kgCO2`
- `Steps_Walked`
- `Calories_Burned`
- `Sleep_Hours`
- `Social_Media_Hours`
- `Public_Events_Hours`

---

## 🎯 Objectives

- Analyze carbon footprint behavior
- Study relationship between transport and emissions
- Understand energy consumption patterns
- Examine lifestyle-health correlations
- Identify high environmental impact citizens

---

## 📈 Key EDA Analysis Performed

### 1️⃣ Data Cleaning
- Checked missing values
- Verified data types
- Generated statistical summaries

### 2️⃣ Transport Analysis
- Distribution of transport modes
- Transport mode vs Carbon Footprint

### 3️⃣ Energy Consumption Analysis
- Distribution of home energy usage
- Energy consumption vs CO₂ emissions correlation

### 4️⃣ Health & Lifestyle Analysis
- Steps vs Calories burned
- Work hours vs Sleep hours

### 5️⃣ Sustainability Insights
- Charging station usage trends
- High carbon footprint citizen identification
- Correlation heatmap

---

## 📊 Key Insights

- 🚗 Private transport users show higher carbon emissions.
- ⚡ Higher home energy consumption increases carbon footprint.
- 🚶 Steps walked strongly correlate with calories burned.
- 😴 Increased work hours reduce sleep duration.
- 🔋 EV charging usage indicates growing sustainable adoption.
